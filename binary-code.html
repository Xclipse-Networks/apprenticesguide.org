Digital Fundamentals

Binary Code.

In the world of electronics and PLCs, binary code is the fundamental language machines use to communicate. Everything from logic gates to microprocessors relies on this base-2 system.

Electrical States

Binary is a base-2 number system using only two digits: 0 and 1. In industrial control circuits, these digits map directly to voltage levels.

0
State: OFF No Voltage (0V DC)
1
State: ON Presence of Voltage (24V DC)

PLC Ladder Logic Representation

Logical Addressing

In PLC programming, binary states are used in Ladder Logic to control field devices. Turning on a motor is a process of writing a logical 1 to a specific output address.

Bit-Level Control

Each physical terminal on a PLC corresponds to a single bit (0 or 1) in the processor memory.

Core Takeaway

Understanding how binary translates to real-world signals is essential for automation, control circuits, and digital electronics.
